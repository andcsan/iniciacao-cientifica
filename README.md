# Iniciação Científica

## Contagem Automática de Células de Peixes com Micronúcleos

Projeto de iniciação científica desenvolvido em python.

## Scripts

```shell
blob.py
python blob.py <imgloc> <cortedir>

checagem-celula.py
python checagem-celula.py <dados> <posicao>

database.py
python database.py <posdir> <negdir> <saveloc>

default-net.py
python default-net.py <in> <hidden1> <hidden2> <out> <nome>

folds.py
python folds.py <dadosloc> <savedir> <nfolds>

offset.py
python offset.py <imgloc> <cortedir> <corte_tam> <offset>

run2.py
python run2.py <img_dir> <rede_dir> <corte_tam> <offset>

training.py
python training.py <data_loc> <default-net> <save_loc>

training-stratified-cross-validation.py
python training-stratified-cross-validation.py <netloc> <dadosloc> <foldloc>
```
